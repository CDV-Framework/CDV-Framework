## Notes for Template Users

**How to Use This Template:**

1. **Save a copy** for your case study
2. **Fill in all sections** - don't skip even if "no data"
3. **Provide evidence** for every score
4. **Document sources** meticulously
5. **Be honest** - include negatives
6. **Compare** with at least 2 similar platforms
7. **Get peer review** before publishing

**Minimum Required Sections:**
- All CDV dimension scores with rationale
- Evidence & Data
- Lessons Learned
- References

**Optional but Recommended:**
- Comparative Analysis
- Recommendations
- Appendices

**File Naming:**
`[Platform-Name]-[Country]-CDV-Case-Study-[Year].md`

Example: `Decidim-Barcelona-Spain-CDV-Case-Study-2025.md`

**Contribution:**
Once completed, submit as Pull Request to CDV-Framework repository.
